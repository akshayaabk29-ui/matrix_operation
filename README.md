Matrix Operations in C
Overview

This project is a Matrix Operations Calculator developed using the C programming language. It allows users to perform common matrix operations such as addition, subtraction, multiplication, and transpose. The program uses two-dimensional arrays, nested loops, and switch-case statements to process matrix calculations.

This project is ideal for students and beginners learning data structures, matrices, and basic programming concepts in C.

Operations Performed
1. Matrix Addition

Adds corresponding elements of two matrices having the same dimensions.

2. Matrix Subtraction

Subtracts corresponding elements of two matrices having the same dimensions.

3. Matrix Multiplication

Multiplies two matrices using row-column multiplication rules.
4. Matrix Transpose

Converts rows into columns and columns into rows.

Algorithm
Start the program.
Display matrix operation menu.
Read user choice.
Input matrix dimensions and elements.
Perform the selected operation.
Display the resulting matrix.
Repeat if required.
End the program.
